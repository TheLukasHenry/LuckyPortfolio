Code

I used javascript for loop to make css grid playground. I put all grids to one array and made another array representing the snake. I made control function for snake direction and hooked it to the arrow keys. Another function was to create snake food. The biggest move function deals with loss scenarios, uses Timeout to move snake, deals with snake eating food and food's effect. I gave snake 3 colors with for function and hooked it to necessary event listeners.
